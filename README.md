# DevOps Project – End-to-End CI/CD Pipeline

## Overview
This project demonstrates a complete DevOps workflow, covering application build, containerization, continuous integration, continuous deployment, and monitoring.

The goal is to simulate a real-world production pipeline using industry-standard tools and best practices.

---

## Architecture
- Source Code Management: GitHub
- CI/CD Pipeline: Jenkins
- Containerization: Docker
- Container Orchestration: Kubernetes
- Infrastructure: AWS (EC2 / EKS)
- Configuration Management: Ansible
- Monitoring & Observability: New Relic / Prometheus / Grafana
